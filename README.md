# Método Abreu Performance

Sistema web premium em Flask para personal trainer, com área administrativa e área do aluno responsiva.

## Como executar

```powershell
cd metodo_abreu_performance
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Abra no navegador:

http://127.0.0.1:5000

## Login demo

- Administrador: use um e-mail contendo `admin` ou `personal`.
- Aluno: use qualquer outro e-mail.

## Telas principais

- Landing page
- Login
- Dashboard do personal
- Lista de alunos
- Perfil completo do aluno
- Treinos
- Financeiro
- Área mobile do aluno
- Treino do aluno
- Evolução
- Financeiro do aluno
- Perfil mobile
